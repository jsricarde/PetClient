# PetClient

Petclient is a mobile client application development in ionic framework, is a part of Petime platform.

Prerequisites:
  - node.js and npm installed.
  - bower installed.
  - ionic and cordova installed.



### Version
1.0.0

### Installation

PetClient requires [Node.js](https://nodejs.org/) v4+ to run.

You need Ionic and Cordova installed globally:

```sh
$ npm install -g ionic
```
Download the git. repository
```sh
$ git clone git@github.com:jsricarde/PetClient.git
```
Install the node_module and bower dependences

```sh
$ cd PetClient
$ npm install
$ bower install
```

### Usage

Issue the command `ionic serve --lab` in your project's directory.

```sh
$ 
$ npm install
$ bower install
```
